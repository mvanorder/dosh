Digital Ocean Shell
===================

CLI for Digital Ocean using [digitalocean-api](https://github.com/valerylisay/digitalocean-api/).

How To Install
==============

clone with git:

	git clone https://github.com/mvanorder/dosh.git

Then create ~/.dosh.cfg with the following:

  ```
  [digitalocean]
    token = <your API token>
  ```
